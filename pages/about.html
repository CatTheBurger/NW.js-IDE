NW.js IDE

Бесплатная кроссплатформенная среда программирования и разработки приложений и игр. Поддерживаемые языки и типы файлов: JavaScript, JSON, HTML, CSS, PHP.

NW.js

Бесплатная среда для создания окружения, позволяющего запускать JavaScript приложения на десктопе (Windows, Linux, MacOS) с возможностью сборки проекта, как цельного приложение.

Дерево файлов

  • - Папка
  • - *.js | файл JavaScript
  • - *.json | файл JSON
  • - *.html | файл HTML
  • - *.css | файл CSS
  • - *.php | файл PHP
  • - *.[exp]+ | собираемый файл Joint (цвет иконки зависит от расширения [exp])


Горячие клавиши редактора кода

  • Ctrl+S - Сохранить файл
  • Ctrl+F - Поиск по файлу
  • Ctrl+H - Поиск и замена
  • Ctrl+D - Удалить выделенное (или строку)
  • Ctrl+Shift+Up (Alt+Up) - Передвинуть выделенное (или строку) вверх
  • Ctrl+Shift+Down (Alt+Down) - Передвинуть выделенное (или строку) вниз
  • Ctrl+Enter - Запуск проекта
  • Ctrl+Click - Установить дополнительный курсор ввода
